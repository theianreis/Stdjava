## Conceitos Iniciais de POO
- Objetos: Instância de uma classe.
- Atributo: Váriavies definidas dentro da classe.
- Classe: Molde para criar um objeto.
- Método: Função definida dentro da classe que criar restrições para o objeto.
- Construtor: Método que criar um objeto para iniciar um atributo da classe e preparar o objeto para uso.
- Getters: Acessador
- Setters: Modificador
- Estado: Atribuições para o atributo.
- Escopo: Visualização de variável 
- Passagem de parâmentros: Conversa de objetos

## Tipos de variáveis
- Variável de instância: Pertence ao obj
- Variável local: Existe dentro do método.
